### 근막부(A상부승모근, B장요근 압박이완 -> 이완강화)

```
A 상부승모근에 대한 압박/이완 기법, 이완/강화 기법을 연속적으로 시행하세요

"머리를 (오른쪽으로) 이쪽으로 천장보고 누으시고요"
좌측을 환측이라 가정하고 시행하겠습니다. 

먼저 압박이완기법은, 유양돌기를 타고 내려와 촉지되는 근벨리의, 외측1/3지점에서 결절을 찾고
보조수로 [견관절 굴곡 -> 외전 -> 외회전]시키면서 POC를 찾습니다. 
"숨들이 마쉬고 숨참아볼게요. 12345.  숨내쉬고" 4초간 기다렸다가 다시 3~4회 정도 반복해서 압박을 실시하고
수동적으로 중립자세로 돌아와 재평가를 실시해서 결절이 소실되었는지 확인합니다. 

이완/강화 기법은, 
(오른손)주동수로 엄지를 벌려서 후두부를 잡고, 보조수로 쇄골1/3지점에서 어깨를 가볍게 눌러 고정한 다음,
[굴곡 -> 건측 측굴]한 뒤, 회전방향에 따라서
건측은 후부 / 중간은 중부 / 환측은 전부섬유를 치료하는데,
편의상 전부섬유를 치료한다고 가정하고,
제한가동범위를 확인한 상태에서 측굴을 풀어 중립범위로 돌아온 뒤,
환자에게 "환자분 왼쪽어깨와 머리가 가까워 지도록 부분을 힘줄꺼에요."
"숨들이 마쉬고 숨참고, 힘주세요 -> 1234567 -> 힘빼고 숨내쉬고" 
4초간 휴지기후 -> [ 더 굴곡 측굴 회전]으로 스트레칭하고 -> 새로운 제한가동범위을 찾아 3~4회 반복합니다. 
```

### 골반부(A우측천골 전방염전-평가 정골, B좌측천골 후방염전-평가 근막기법)

```

1. <쉬우니까 공간확보먼저> 환측 하지를 건측발목에 걸어줘서, 후방으로 나올 공간을 확보하고

3. "환자분 숨 크게 들여 마쉬고 숨내쉬면서 ~ " 제한가동범위까지 가서, 회전을 섞어 + 체중을 실어 교정합니다.
cf) 쓰러스트 4개 : 체간부 - L4 ERSL 측와위 교정 / 골반부 - 우측천골 전방염전 측와위 교정 / 두경부 - C3 FRS r 왼쪽관절돌기 닫을 때 교정 / 사지부 - 앙와위 흉쇄관절 교정기법 : 쇄골의 후방 활주 제한
```

### 체간부 (A흉추 T8 FRSr 평가 시술, B요추 L4 ERL 평가 시술 - 후방염전천골을 45빼고, 밀고 vs 요추L4는 굴곡, 굴곡) 
- 흉추는 MET를 2번(신전좌회전좌측굴 만들어MET -> 장근부교차 신전MET)한다)

```

1. <양팔을 교차시키고> 보조수로 어깨를 잡은 상태에서 
2. "환자분 오른쪽 엉덩이가 앞으로 나간다는 느낌으로, 가슴들면서 배를 내밀어 보실게요" 하면서 좌회전 좌측굴을 만든다.
3. "환자분 어깨를 드는 방향으로 힘줄거에요" 하면서
"숨들이 마쉬고 숨참고, 힘주세요 -> 1234567 -> 힘빼고 숨내쉬고"

2단계: 장근부교차 신전 MET 

굴곡 변위를 치료하기 위해, 주동수 장근부로 T8아랫분절을 접촉후 배를 충분히 내밀어 [E]를 만들어준 상태에서 
"환자분 숙이는 방향을 힘줄거에요"
"숨들이 마쉬고 숨참고, 힘주세요 -> 1234567 -> 힘빼고 숨내쉬고" 
4초간 휴지기후 -> [더 배내밀고] 새로운제한장벽을 찾으면서 스트레칭하여 2~3회 반복합니다. 
```

### 두경부(A후두과(E RL SR) 진단 근막기법, B경추 C3 FRSr 2진단 근막 정골기법)
