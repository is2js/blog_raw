VO, DTO

  • Dto는 controller에 일단 넣어준다.

  • ROUND는 hasNext패턴으로 줄어가는데, 생성자 검증은 1이상 <-> 1씩 줄여가니 0까지는 감.
    • 생성자에서 검증하되, 새객체 반환은 검증없는 private 기본생성자이용
