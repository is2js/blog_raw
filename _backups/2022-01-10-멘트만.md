### 근막부(A상부승모근, B장요근)

```
A 상부승모근에 대한 압박/이완 기법, 이완/강화 기법을 연속적으로 시행하세요

"머리를 (오른쪽으로) 이쪽으로 천장보고 누으시고요"
상부승모근은 누칼라인~항인대~경추1-7극돌기에 붙어서 -> 견봉~견봉단~쇄골외측1/3지점까지 붙는 근육인데
좌위에서 양측 단축검사를 먼저하고 단축된 쪽에서 결절을 찾아 압박/이완는데, 좌측을 환측이라 가정하고 시행하겠습니다. 

압박/이완 기법은, 유양돌기를 타고 내려와 촉지되는, 근벨리 외측1/3지점에서 결절을 찾고
보조수로 [견관절 굴곡 -> 외전 -> 외회전]시키면서 POC를 찾습니다. 
"숨들이 마쉬고 숨참아볼게요 12345  숨내쉬고" 4초간 기다렸다가 다시 3~4회 정도 반복해서 압박을 실시하고
수동적으로 중립자세로 돌아와 재평가를 실시해서 결절이 소실되었는지 확인합니다. 

이완/강화 기법은, (오른손)주동수로 엄지를 벌려서 후두부를 잡고, 보조수로 쇄골1/3지점에서 어깨를 가볍게 누른 다음 
[굴곡 -> (대측)건측 측굴]한 뒤, 
후부섬유는 건측 회전 / 중부섬유는 중간 / 전부섬유는 환측 회전하여 제한장벽을 찾는데, 편의상 전부섬유를 교정한다고 가정하고
굴곡 -> 건측측굴 -> 환측회전 후, 측굴을 풀어 중립범위로 돌아온 뒤,
환자에게 "환자분 왼쪽어깨와 머리가 가까워 지도록 부분을 힘줄꺼에요."
"숨들이 마쉬고 숨참고, 힘주세요 -> 1234567 -> 힘빼고 숨내쉬고" 
4초간 휴지기후 -> [ 더 굴곡 건측측굴 환측회전]으로 새로운제한장벽을 찾으면서 스트레칭하여 3~4회 반복합니다. 
```

```

"머리를 (오른쪽으로) 이쪽으로 천장보고 누으시고요"

"환자분 무릎을 천장으로 들어볼거에요"
"숨들이 마쉬고 숨참고, 힘주세요 -> 1234567 -> 힘빼고 숨내쉬고" 
4초간 휴지기후 -> [무릎을 더 눌러으로] 새로운제한장벽을 찾으면서 스트레칭하여 3~4회 반복합니다.
```

### 골반부(A우측천골 전방염전, B좌측천골 후방염전)

- B후방염전은 좌회전우측굴-측와위3개이용 MET)

```

***"머리를 (왼쪽으로) 이쪽으로 엎으려서 누워보세요"
우측천골 전방염전은, 천골이 우회전+좌측굴되어 좌측천골기저부가 앞으로 찝힌 변위인데, 
우경사축에 대한 좌측의 변위이므로 -> 6앉은자세 굴곡검사에서 왼쪽이 양성으로 나올 것입니다.(필요시 역SLRT도 같이 시행하면, 전방염전되쪽이 10cm 덜 하지거상 됩니다.)
여기서는 PSIS내측과 천골ILA를 접촉하여 4점 테스트를 하면, 들숨시 좌측천골기저부의 신전움직임이 떨어짐을 확인할 수 있습니다. 

정골기법을 시행하기 위해, 환측 하지를 건측발목에 걸어줘서, 후방으로 나올 공간을 확보하고
건측 천골기저부와 하외측각에 연조직견인으로 접촉한 뒤, 
"환자분 숨 크게 들여 마쉬고 숨내쉬면서 ~ " 제한장벽까지 회전시키면서 체중을 실어 교정합니다.
cf) 쓰러스트 4개 : 체간부 - L4 ERSL 측와위 교정 / 골반부 - 우측천골 전방염전 측와위 교정 / 두경부 - C3 FRS r 왼쪽관절돌기 닫을 때 교정 / 사지부 - 앙와위 흉쇄관절 교정기법 : 쇄골의 후방 활주 제한
```


### 체간부 (A흉추 B요추) 
- 흉추는 MET를 2번(신전좌회전좌측굴 만들어MET -> 장근부교차 신전MET)한다)
- 요추는 3손가락으로 잡는다. F RS R을 만들어 나간다.


```
A 흉추 8번의 굴곡 우측굴 우회전 변위를 평가하고 시술하시오.(T8 FRSr)

평가를 위해서, 견갑골 하단에서 2지로 T8극돌기를 찾고, 3.5cm외상방에서 T8횡돌기를 촉지할 수 있습니다. 
보조수로 어깨를 잡은 상태에서 "가슴을 드는 느낌으로 배를 앞으로 내보세요"하여 신전시켰는데도, 우측횡돌기가 돌출되어있다면, FRSr로 평가할 수 있습니다.

1단계: 신전+좌회전+좌측굴 MET 

시술을 위해 T8에 [E RSL, 좌회전좌측굴]을 만들어줘야하므로, 
1. <양팔을 교차시키고> 보조수로 어깨를 잡은 상태에서 
2. "환자분 오른쪽 엉덩이가 앞으로 나간다는 느낌으로, 가슴들면서 배를 내밀어 보실게요" 하면서 좌회전 좌측굴을 만든다.
3. "환자분 어깨를 드는 방향으로 힘줄거에요" 하면서
"숨들이 마쉬고 숨참고, 힘주세요 -> 1234567 -> 힘빼고 숨내쉬고" 
4초간 휴지기후 -> [더 배내밀고 + 좌회전 + 좌측굴을 만들어] 새로운제한장벽을 찾으면서 스트레칭하여 2~3회 반복합니다. 


2단계: 장근부교차 신전 MET 

중립위로 와서, 교차된 팔을 잡고, 주동수 장근부로 T8아랫분절을 접촉후 배를 충분히 내밀어 [E]를 만들어준 상태에서 
"환자분 숙이는 방향을 힘줄거에요"
"숨들이 마쉬고 숨참고, 힘주세요 -> 1234567 -> 힘빼고 숨내쉬고" 
4초간 휴지기후 -> [더 배내밀고] 새로운제한장벽을 찾으면서 스트레칭하여 2~3회 반복합니다. 
```

