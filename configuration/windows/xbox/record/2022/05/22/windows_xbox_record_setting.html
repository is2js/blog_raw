windows xbox 녹화 단축키

  1. win + alt + R : 빠른 녹화

  2. win + g : xbox 녹화 설정

크롬 소리 녹화가능하게 하기

  • 게임+마이크 기반으로 녹화되는 것이 기본 설정이다

  • 아래 그림과 같이 설정을 바꿔야한다.

  1. win + g를 눌러 xbox 기능으로 들어간다.
  2. 우측 상단의 톱니바퀴 > 캡처 중으로 들어간다
  3. 아래 2가지를 체크한다.
    • 게임 플레이 중 백그라운드 녹화
    • 게임(게임, 사용가능 마이크) 옵션을 -> [x] 모두(게임, 사용가능 마이크, 앱, 시스템)로 변경한다
